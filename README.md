# Simulation-Based Analysis of Promotional Strategies in Retail Sales

# Overview
This project analyzes the impact of different promotional strategies on retail sales using Machine Learning and simulation techniques.
A Random Forest Regression model was developed to forecast sales and evaluate promotional effectiveness under multiple business scenarios.

# Objectives
- Analyze retail sales patterns using EDA
- Measure the impact of promotions on sales
- Build a predictive sales forecasting model
- Simulate different promotional strategies
- Generate data-driven business recommendations

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

# Machine Learning Model
- Random Forest Regression
- RMSE ≈ 1406
- R² ≈ 0.796

# Key Findings
- Strong promotions increased predicted sales by 58.25%
- Targeted promotional strategies achieved performance similar to regular promotions
- Promotional activities significantly impact retail sales
